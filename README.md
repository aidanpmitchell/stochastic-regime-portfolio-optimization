# Stochastic Regime-Switching Portfolio Optimization

This project explores how stochastic processes and portfolio optimization can be combined to build a regime-aware investment strategy.

## Main Ideas

- Model market regimes using stochastic processes / Markov chains
- Estimate expected returns and covariance matrices
- Solve mean-variance portfolio optimization problems
- Backtest dynamic allocation strategies across market regimes
